# NDC Notice Bot

Fetches new notice from https://ndc.edu.bd/ and automatically sends it to all users that have added the bot.

## Add the bot
- [@NDCNoticeBot](https://t.me/NDCNoticeBot)

## Usage
- /start - Adds you to users list
- /last - gives you last the notice

Libraries: 
- pyTelegramBotAPI
- requests
- selectolax or bs4
- python-dotenv
